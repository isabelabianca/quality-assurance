ISO 25010 - Critérios usados para avaliar o software (características de qualidade)
	-> Funcionalidade
		-> Atendimento de requisitos (product backlog)
		-> Facilidade de configurar
		-> Precisão de cálculos
		-> Segurança de dados
		-> Funcionamento integrado de componentes/módulos
	-> Portabilidade
		-> Facilidade de instalação
		-> Facilidade de substituição
		-> Facilidade de adaptadção a novas plataformas
	-> Manutenabilidade - facilidade de manutenção
		-> Facilidade de análise
		-> Facilidade de mudança
		-> Facilidade de teste
	-> Confiabilidade
		-> Facilidade de se recuperar de falhas
		-> Proteção contra falhas
		-> Estabilidade de processamento e resultados (mantém um comportamento, gerando o mesmo resultado)
	-> Usabilidade - facilidade de uso
		-> Facilidade de compreensão
		-> Facilidade de aprendizado
		-> Facilidade de operação
	-> Desempenho
		-> Tempo consumido
		-> Recursos consumidos
		
ISO 12207 - Fatores influenciadores da qualidade - Processo e controles de trabalho a fim de atingir as expectativas do consumidor 
	    (é o foco se tratando de desenvolvimento da qualidade)
	-> Competências individuais
	-> Processo e controles de trabalho
	-> Ferramentas e materiais
	Tipos de Processos:
		-> Principal
			-> Produção, teste, modelagem de BD e documentação - o que é essencial
		-> Organizacional
			-> Planejamento, tudo o que é relativo ao gerenciamento do projeto
		-> Suporte
			-> Atividades administrativas ou técnicas não relacionadas diretamente à produção 
			   do software (infraestutura), mas que influenciam totalmente nela
